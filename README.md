# Electron Tasks App

A basic Electron + React + TypeScript app that keeps task state in the main process and pushes updates to the renderer.

## Scripts

- `npm run build`
- `npm run lint`
- `npm run format`
- `npm run test`

After making changes, run the scripts above to ensure the app builds cleanly and stays formatted and tested.

## Run the app

1. `npm install`
2. `npm run build`
3. `npm run start`
